## 🐹 프로젝트 개요

뽀모도로 앱

- 재생 / 일시정지 버튼
- 25분 카운트 다운
- 25분 완료시, 1 Round 카운트 업
- 4 Round 완료시, 1 Goal 카운트 업

